# WaterMark_Images

> Agregar una marca de agua a imagenes dentro de un directorio.



Este código crea una pequeña marca de agua en la esquina inferior derecha a un grupo de fotografías.

## Usage example

Se puede guardar imagenes en el directorio y se agregará una marca de agua.
